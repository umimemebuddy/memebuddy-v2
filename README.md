# MemeBuddy

No bullshit. Just booze and PVP. Cocktail recipes meets crypto trading.

## Tech Stack

- Next.js 16 (static export)
- Tailwind CSS v4
- TypeScript
- Deployed on Cloudflare Pages

## Development

```bash
npm run dev     # Start dev server
npm run build   # Build static export to ./out
```

## Deploy

```bash
npx wrangler pages deploy out --project-name=memebuddy-v2
```
